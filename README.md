# gebhart.dev

A general purpose website serving as the primary location of my portfolio, contact information, writings, and general information about myself. Built with Markdown and Sakura, powered by Zola and Netlify.

## Building

You'll need [zola][1] installed.

To run:
`zola serve`

## Deleting posts

If a post/page is to be deleted, it's URI needs to be added to `_redirects` in order to accurately response with a permanent redirect.
